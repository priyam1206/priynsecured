**PRIYN Secured** is a professional-grade, high-performance local encryption tool. Part of the **PRIYN** ecosystem, **Secured** is built on a zero-trust, offline-first philosophy—ensuring your private files stay on your hardware, never the cloud.

---

## ✨ Key Features

*   **Bank-Grade Encryption:** Uses AES-256-GCM streaming encryption to secure massive workloads and large directories instantly.
*   **Hardware-ID Locking:** Your vault is cryptographically bound to your specific PC. Data remains impenetrable even if moved to another device.
*   **Zero-Knowledge Privacy:** Features automatic filename masking and active memory wiping post-execution to prevent RAM scraping.
*   **Tamper Protection:** Built-in integrity monitoring detects external file modifications and triggers an instant lockdown.
*   **100% Offline:** Zero external server pings, zero tracking, and zero account requirements.

## 🚀 Getting Started

1.  **Download:** Get the latest `secure.exe` (49.5 MB) from the [Releases](https://github.com/priyam1206/priynsecured/releases) page.
2.  **Launch:** Run the standalone executable. No installation is required.
3.  **Secure:** Select your target folder and set your master key.
4.  **Done:** Your data is now secured by PRIYN.
